PROJECT SPECIFIC REQUIREMENTS PROJECT SPECIFIC REQUIREMENTS PROJECT SPECIFIC REQUIREMENTS PROJECT SPECIFIC REQUIREMENTS 
A complete integrated control system based on a Metso DNA DCS will be provided 
for data acquisition, regulatory control, discrete logic processing, alarms, current 
trends, historical trends, event logs, reports and historical data archiving. Motors will 
be controlled using Profibus technology and Smart Starter. All the analog and digital 
signals will be controlled by the DCS without the use of a separate PLC, with the 
exception of the turbine generator and other vendor supplied packages where a 
control system is furnished. A turbine generator serial / Ethernet communications 
interface shall be provided that will support communications between the Plant 
Process DCS and the Siemens supplied PCS7 DCS for Turbine Generator control. 
AMEC I&C design responsibilities include supplying Instrument Index, Instrument 
Specifications, Loop Diagrams, Installation Details, Location Drawings and Cable 
Schedule. Additionally, Logic Diagrams and Complex Loop Narratives will be provided 
to assist in DCS Configuration.

Control System for the following areas:

Waste Water Treatment.
